### C언어

## 비트연산자

- '<<' : 왼쪽으로 이동
- '>>' : 오른쪽으로 이동

예를 들어 5>>2 라면 5의 이진수인 101(2)에서 오른쪽으로 2칸 이동해서 1만 남게 되어 5>>2 는 1이 된다.
반대로 5<<2 라면 왼쪽으로 두칸 이동하면서 10100(2) 즉 20이 된다.

### C언어

- '*' : 자료형 *변수명; // 포인터의 변수 선언을 할때 사용한다.
- '&' : 주소를 반환하는 연산자
- int *ptr = &arr[0]; 과 int*ptr = arr; 은 동일한 문장이다.
